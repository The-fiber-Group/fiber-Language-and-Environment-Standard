The International Standard for

ASC: the Algebraic System for Computation

Revision: 0.0

Abstract

  1. This document specifies the form, interpretation, and execution of programs written in the programming language, ASC. The purpose of this document is to provide a common basis for the portability and maintainability of ASC programs between implementations.
