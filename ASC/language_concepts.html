Language Concepts

  1. An ASC program consists of named entities which contain some information and are associated with a type entity.

  2. The name of an entity determines the contexts within the program where that entity will be accessible.

  3. The type of an entity determines the range of information or values that can be held by an entity and what operations are available for that entity.

  4. Each entity can be classified as either variable or constant which determines if the information contained by the entity can be changed by the program.

  5. The ASC language provides primitive entities which can be used to build a larger program. They are...

    • variables
    • primitive data types
    • compound data types
    • abstract data types
    • continuations
    • functions
    • modules
    • syntax extensions
    • symbols
    • expressions
