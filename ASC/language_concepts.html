Language Concepts

Data Objects

  1. An ASC program is executed through the manipulation of data objects. A data object is a construct which stores values and has both a representational structure and a precision. The representational structure affects how values are stored. The precision affects the range of values that are possible to store.

  2. All data objects shall be associated with a single name and data type for the duration of their existence.

  3. There are two classes of objects, variable objects and descriptive objects, which describe how the objects are used in the program.

  4. All data objects are declared. Declaring an object associates it with a name and a data type.

  5. In the case of descriptive object, the definition expression may contain additional information alongside the name and data type.

Variable Objects

  1. Variable objects are used as...

    • program variables
    • program constants
    • continuations

  2. Variable objects are mutable and contain data that may be transformed by the program being executed.

  3. Variable objects have an implicit property of duration where the data they contain is accessible for a static duration.

Descriptive Objects

  1. Descriptive objects are used to define the effects of a program. The contents of descriptive object may only be read. Descriptive objects are used as...

    • primitive data types
    • abstract data types
    • compound data types
    • functions
    • modules
    • syntax extensions
